Pasta is a tiny SASS file that provides a great starting place for web projects. It combines a solid reset and light styling for common web elements, yummy.

Getting Started

Pasta is a collection of partial SASS files, broken out into their core functions. Individual files can be found in the _ingredients folder within the main Pasta directory.

  • _water.scss contains all the variables and mixins used throughout Pasta. This is most likely the file you’ll edit the most as you fine-tune styles specific to your project.
  • _flour.scss establishes a solid reset for core HTML elements and is technically optional. If you prefer a more common reset like Normalize, feel free to comment out flour from the main _pasta.scss file.
  • _eggs.scss starts to build on the core foundation for the light cross-browser styling associated with Pasta.

OPTIONAL CONFIGURATION

If all you want to do is adjust basic variables but not core stylings, you can include the declarations before you import Pasta into your main stylesheet:

$type-sans: 'Open Sans', sans-serif;
$type-serif: 'Lora', serif;
$type-mono: 'Source Code Pro', monospace;
$color-base: #523A00;
$color-highlight: #F2AE00;

@import "pasta/pasta";

After importing Pasta into your main stylesheet, you can use any variables or mixins throughout your main stylesheet, easily building on top of _eggs.scss as you establish the look and feel of your project.
