Pasta

Simply Delicious

Pasta is a tiny SASS file that provides a great starting place for web projects.

It combines a great reset and light styling for common web elements, yummy.

Typography

Easily control site-wide typography using simple variables, it's as easy as setting a numeric value for font-size. Want to control font-size and line-height for any element—there's a mixin for that.

When the main .wrapper element reaches a width greater than 1024px, styled ul and ol elements adapt a hanging punctuation format to better align to corresponding text.

Form Elements

Form elements are lightly styled for cross-browser consistencies. Radio buttons and checkboxes should be wrapped in .radio and .checkbox classes respectively. Additionally, if you apply the class .inline to either input group, elements will adapt an inline style for improved legibility (as seen above).

The grid system is responsive by default applying rational widths when space permits. For viewports 480px and below .grid-1-4 and .grid-1-6 fill partial widths of the parent element and all other .grid elements are full-width. Additionally, by applying .grid-nest to a .grid element, you can nest grids of varying widths inside, coming up with new widths not already accounted for, yay math!
